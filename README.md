# Demo Shop
***

This is demo version of a site for a shop, using Strapi.io backend API + Nuxt.js frontend.

## How to run
in backend directory 
``` shell
strapi dev
```
in frontend directory 
``` shell
yarn run dev
```
It will open front on http://localhost:3000/
